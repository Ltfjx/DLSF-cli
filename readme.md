# DLSF-cli
DLSF-cli is a command-line interface for [DLSF](https://github.com/Ltfjx/DLSF).  

> 桜が咲いたら忘れましょ、 
> 咲かない花は忘れましょ。

## Install
```
npm install -g dlsf-cli
```

## Start
Run the following command in your terminal to get help:  
```
dlsf-cli
```
